# 🎾 Pyeongchon ACE Ranking System

An integrated match management solution designed for **Pyeongchon ACE**, a premier tennis club based in Anyang, Gyeonggi-do, South Korea. This system fosters a fair and competitive environment through data-driven ranking and automated tournament management.

## ✨ Key Features

* **Smart Entry & Matchmaking:** Streamlined participant registration and automated draw generation for seamless tournament starts.
* **Result Tracking:** Intuitive interface for recording match scores and maintaining a comprehensive historical database.
* **Dynamic ELO Rating System:** Sophisticated ranking management based on the ELO rating algorithm, providing an objective measure of player skill levels.
* **Insightful Analytics:** Generates engaging performance reports, including:
    * **Partner Stats:** Identify your "Dream Partner" with the highest win rates.
    * **Nemesis Tracking:** Discover your "Nemesis" who has taken the most ELO points from you.
    * **Achievement Badges:** Earn special titles like "Bagel Master" or "Iron Man."
* **Interactive Dashboard:** Real-time ranking updates and session-based result summaries to boost club engagement.

## 🛠 Tech Stack
- **Frontend:** React / Next.js (Deployed on Vercel)
- **Database:**  Firebase 
- **Language:** JavaScript / TypeScript
