🏸 ACE 랭킹 시스템 사용 가이드

📋 1. 참가자용 가이드 (모든 사용자)

✅ 참가 신청 (참가신청 탭)
  • 대기: 관리자가 회차를 오픈할 때까지 기다립니다.
  • 신청: 접수가 시작되면 본인의 이름을 입력하고 신청합니다.
  • 확인: 하단 명단에 본인의 이름이 나타나면 정상 처리된 것입니다.
⚔️ 대진표 및 경기 진행 (대진표 탭)
  • 대진 확인: 본인이 속한 조(A, B 등)를 선택하여 상대를 확인합니다.
  • 점수 입력: 경기가 끝나면 자신들의 경기결과를 입력합니다.
📊 랭킹 및 대시보드
  • 종합 랭킹: 현재 모든 선수의 ELO 점수, 승률 등을 확인합니다.
  • 도토리 성장 추이: 대시보드에서 본인의 이름을 선택하여 실력 변화 추이를 확인하세요.

🔐 2. 관리자용 가이드 (운영진)

🔑 관리자 로그인

상단 [관리자 로그인] 버튼 클릭 후 암호 입력 시 관리 기능이 활성화됩니다.

📂 DB 관리
  • DB 전환: 드롭다운 메뉴에서 활성화 할 DB를 선택합니다.
  • 새 DB 생성: 새로운 이름을 입력하고 생성합니다. 시즌 재시작시 사용합니다.
⚙️ 대회 운영 프로세스
  1. 회차 오픈: 참가신청 탭에서 회차 번호를 확인하고 [참가 접수 시작]을 누릅니다.
  2. 명단 관리: 신청 명단은 랭킹 순으로 자동 정렬되며, 이름 옆에 현재 순위가 표시됩니다.
  3. 대진표 생성: [대진표 생성] 클릭 시 신규 참가자는 랭킹권 아래에 무작위 배치됩니다.
  4. 랭킹 확정: 경기 종료 후 [랭킹전을 종료합니다]를 누르면 결과가 반영됩니다.
💾 데이터 활용
  • CSV 내보내기: 히스토리 탭의 [CSV 내보내기] 버튼을 통해 전체 경기 기록을 엑셀 파일로 저장할 수 있습니다.

📊 포인트 산정 방식 (ELO Rating)

공정한 실력 측정을 위해 **ELO 레이팅 시스템**을 사용합니다. (초기 점수 1500점)

1. 경기 결과별 포인트 변동

구분 점수 변동 예시 (기대승률 50%) 비고
승리 점수 상승 (+) +16.0 상대와의 실력 차이에 따라 변동폭 결정
완승 (6:0) 점수 상승 (×1.5) +24.0 승리 점수의 50% 추가 보너스
무승부 소폭 변동 ±0 강팀은 소폭 하락, 약팀은 소폭 상승
패배 점수 하락 (-) -16.0 승리 팀이 얻은 만큼 점수 차감

2. 추가 규칙

  • 완승 보너스: 6:0으로 승리 시, 승점 변동폭이 **1.5배(50% 추가)** 적용됩니다. (이길 때 더 많이 얻고, 질 때 더 많이 잃습니다)
  • 신규 참가자: 1500점에서 시작하며, 첫 배치 후 실력에 맞는 점수를 찾아갑니다.
